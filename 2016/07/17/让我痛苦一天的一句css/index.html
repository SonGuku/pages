让我痛苦一天的css

看过我主页的兄弟们，应该点击了那个js插件。最开始我的博客名字挡住了插件，每次都点击在

标签上了，别的位置还好就是标签那个位置点击不了插件。有强迫症的我，一定要解决，要不页面就太不丝滑了。

不是专业前端，只浅浅的接触了HTML、css、js。这可怎么办呢，百度的关键字都不知道输入什么。我也不记得搜的是什么了，就看到了pointer-events: none

pointer-events: none意思就是鼠标事件失效的意思。属于css属性，有了它什么连接事件都再见了。一下就击穿了我的blog名字，点击到js插件了。完美解决，哈哈。
